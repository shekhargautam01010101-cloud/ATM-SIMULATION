# 💳 ATM Simulation System (Python Project)

## 📌 Overview
The ATM Simulation System is a menu-driven Python application that mimics the basic functionalities of an Automated Teller Machine. It allows users to perform essential banking operations such as checking balance, depositing money, withdrawing cash, and viewing transaction history.

This project is designed using modular programming concepts and follows a proper package structure for better readability and maintainability.

---

## ⚙️ Features
- 🔹 Check Account Balance  
- 🔹 Deposit Money  
- 🔹 Withdraw Money  
- 🔹 View Transaction History (Mini Statement)  
- 🔹 Menu-driven interface using infinite loop  

---

## 🛠️ Technologies Used
- Python  
- Functions and Modules  
- Basic Data Structures (List, Variables)
